Frequently Asked Questions

What file formats do you support?

We currently support PDF files exclusively for our PDF tools. Supported versions include PDF 1.0 through PDF 2.0, including password-protected PDFs (with password provided), portfolios, and documents with embedded multimedia content. Files can be up to 100MB in size.

How long does processing take?

Processing times vary by operation and file size. Simple operations like metadata viewing are instant, while text extraction typically takes 10-30 seconds. Large file compression or image conversion may require 1-5 minutes. Our system provides real-time progress updates during processing.

Can I use these tools on mobile devices?

Yes, our PDF tools are fully responsive and work seamlessly on smartphones and tablets. The interface adapts automatically to your screen size, and all functionality including drag-and-drop file upload is supported on touch devices.

Do I need an internet connection?

Yes, an active internet connection is required as all processing occurs on our secure servers. This ensures consistent results across all devices and access to powerful processing capabilities without installing software.

How do you handle password-protected PDFs?

For password-protected PDFs, you'll need to provide the document password during upload. The password is used only for document decryption and is never stored or logged. Once processing is complete, the password is immediately purged from system memory.

What happens if I don't download my processed file?

Processed files are available for download for 60 minutes after completion. If not downloaded within this timeframe, all files and processing data are automatically and permanently deleted from our servers for your security and privacy.

What if processing fails?

If processing fails, you'll receive a detailed error message explaining the issue. Common causes include corrupted files, unsupported PDF features, or extremely large documents. Our system automatically cleans up any failed processing attempts and securely deletes associated data.

Can I process multiple files simultaneously?

Some tools like PDF merging support multiple file inputs in a single operation. For other tools, you can open multiple browser tabs to process several files concurrently. Each processing session is independent and secure.

Will processing affect PDF quality?

Most operations preserve original quality completely. Compression tools offer quality settings to balance file size with visual fidelity. Image conversion allows custom DPI settings, and our algorithms are optimized to maintain maximum quality at your chosen settings.

Do you provide technical support?

Yes, we offer comprehensive technical support through our help center and email assistance. Our support team can help with processing issues, feature questions, and technical guidance. Response times are typically within 24 hours for most inquiries.
